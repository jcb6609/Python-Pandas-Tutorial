To install pandas:
    $ pip install pandas

    - General description:
	~ $: command line prompt
	~ pip: Calls Python's standard package manager. It is an online tools that connects to
             an online repository called the Python Package Index (PyPI) to fetch packages.
	~ install: Command that tells pip that we want to downlaod and set up a new package.
	~ pandas: This is the name of the target library. Pandas is an incredibly popular 
                open-source tool used for data manipulation, cleaning, and analysis.
